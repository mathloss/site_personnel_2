Site web personnel pour la gestion de matériel musical

J’ai réalisé ce projet web comme exercice pour mettre en pratique les notions apprises lors du projet "site web pour un restaurant".

L’objectif était de me permettre de réaliser un projet de A à Z plus complexe et plus riche que le premier. Le but était de gérer plusieurs bases de données, de les mettre régulièrement à jour et de les afficher par catégories sur une page web adaptée.

Les impératifs étaient les mêmes que pour le projet "site web pour un restaurant" : simplicité d'utilisation et visibilté sur le web.

Ce projet plus complexe m'a permis d'appliquer les notions apprises au cours de ma formation.
Consulter le code du projet via Github

Technologies utilisées : HTML, CSS, DJANGO, PYTHON, HEROKU
