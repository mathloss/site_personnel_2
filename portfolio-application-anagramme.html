Jeu de l'anagramme

Voilà un petit jeu d'anagramme avec un menu et des boutons pour sélectionner la difficulté (3,4,5... lettres).
J'utilise un fichier texte externe au code pour définir la liste de mots disponibles qui sont triés en fonction du nombre de lettres composant le mot , je casse le mot et je mélange les lettres puis je l'affiche à l'écran.
Le jeu reste à améliorer (liste de mots et interface sont en cours d'amélioration) mais reste fonctionnel.

Technologies utilisées : PYTHON, tkinter, PCLOUD
