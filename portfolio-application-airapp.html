AirApp

Voilà une apllication qui reprend les notions apprises sur les API, tkinter et la production d'un fichier éxécutable.
C'est un programme simple qui permet de vérifier la qualité de l'air d'une zone donnée en entrant le code postale d'une ville qui va acquérir des données générées par une API.
La couleur du fond de la frame change en fonction de l'indice de qualité de l'air, et indique cet indice.
Le but de cet application était d'être capable de récupérer des données Json via une API et de restituer ces données dans un programme avec une interface graphique.
Le projet est disponible en fichier python et en .exe.
L'API utilisée est américaine et donc les codes postaux corresponderont à des villes américaines.Une version française est en prévision ainsi qu'une améloioration de l'affichage.

Technologies utilisées : PYTHON, tkinter, AUTO-PY-TO-EXE, Json, API, PCLOUD
