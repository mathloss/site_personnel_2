Application "Calculatrice"

J’ai réalisé ce projet dans le cadre de ma formation.

L’objectif ? Etre capable de réaliser une application simple avec une interface graphique en utilisant la bibliothèque kivy.

N'ayant jusqu'à présent pas développé d'interface graphique, ce projet m'a permis d'acquérir de nouvelles compétences.

Ce projet simple mais fonctionnel m'a donné l'occasion de confronter le côté code au côté GUI et de créer un lien entre ces deux facettes de la programmation.

Technologies utilisées : PYTHON, KIVY, PCLOUD
